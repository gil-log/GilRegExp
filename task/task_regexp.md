# !([가-힣]+)!is

한글 수집
<br> 
`()` => 수집하겠다.<br> 
`[가-힣]` => 가 문자 부터 힣 문자까지의 문자를<br> 
`+` => 1번 이상 반복되는 문자를

# !(\w\w\w)!is

영문자 나 `_` 문자 세 개 연속 있는 문자 수집<br>

`\w` => 영문자 및 _문자 일치 <br>

`\w`가 세개니까 영문자나 _문자가 세 개 연달아 있는 문자를 수집

# !(\w{3})!is

영문자 나 `_` 문자 세 개 연속 있는 문자 수집<br>

`{3}` => 앞에 있는 문자가 3번 반복되는 문자와 일치
